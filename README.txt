ESPAÑOL:

Este es un pequeño proyecto que hice con Python y DJango (y bootstrap para el CSS)
para poner en práctica mis habilidades y comenzar a crear mi portafolio. El mismo
consiste de tres páginas:
- Inicio: Permite seleccionar los requisitos para generar una contraseña. Se permite
seleccionar requisitos como la cantidad de caracteres (en un rango de 1 a 50), las letras
mayúsculas, los símbolos o caracteres especiales, y los números. Las letras minúsculas
están activadas por defecto y la contraseña se generará solo con ellas aunque no se
seleccionen otros tipos de caracteres. Se redirige a esta página por defecto al ingresar
al mismo por primera vez
- Acerca de: Incluye detalles acerca del autor del sitio (yo), así como enlaces a
mi LinkedIn y GitHub.
- Contraseña: Es el sitio donde se puede ver la contraseña generada con los requisitos
pedidos. Al refrescar la página, se genera una nueva con los mismos requisitos. También
he agregado un botón para copiar la misma al portapapeles.

ENGLISH:

This is a small project I made using Python and DJango, and bootstrap for the CSS to put
on practice my skills and to start creating my own portfolio. The site consists
of three pages:
- Home: Redirected by default to this page when entering. Allows the user to select the
requirements for the randomly generated password. The user can select the number of
characters (in a range from 1 to 50), and can activate the uppercases, the special
characters and the numbers (lowercases are activated by default and the password will be
generated only with those if no other characters are selected)
- About: Contains details about the author (me), and also links to my GitHub and LinkedIn
- Password: Redirected after clicking on the "generate" button in the home page, contains
the generated password. I also added a button to copy it to the clipboard.

my web development skills are better.
